Biblioteca — API

Este es el API para la aplicación de biblioteca localizada en https://library.djangulo.com.

(English version.)

Rutas disponibles:

Nombre Ruta Ejemplo Notas
Listado de libros /books /books —
Libros por id /books/:id /books/11f2a45a-bfc5-41ad-af5d-0e482e84f87b —
Búsqueda de libros por título o autor /books/search?q=<query>&by=<order-by>&order=<asc|desc> /books/search?q=lewis&by=author&order=desc El parámetro q es requerido.
Listado de páginas /pages /pages —
Páginas por id /pages/:id /pages/b80a0417-abfc-4fe9-acda-c2e78edd1161 —
Búsqueda de páginas por parámetros /pages/by-params?book-id=<book-id>&page-number=<page-number> /pages/by-params?book-id=e55dbeb9-d7bb-43b5-b22a-0bf2659d7e42&page-number=1 Ambos parámetros, book-id y page-number son requeridos.
