Library — API

This is the api for the Library app located at https://library.djangulo.com.

(Versión en español.)

Available routes:

Name Route Example Notes
Book list /books /books —
Book by id /books/:id /books/864d0e7c-c309-4186-9737-f20f580a3d3c —
Book search by title or author /books/search?q=<query>&by=<order-by>&order=<asc|desc> /books/search?q=lewis&by=author&order=desc The q parameter is required.
Page list /pages /pages —
Page by id /pages/:id /pages/9a606356-0678-4478-ac4d-ff8bf84634e8 —
Page param search /pages/by-params?book-id=<book-id>&page-number=<page-number> /pages/by-params?book-id=864d0e7c-c309-4186-9737-f20f580a3d3c&page-number=1 Both the book-id and the page-number query params are required.
