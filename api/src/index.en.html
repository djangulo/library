Library — API

This is the api for the Library app located at https://library.djangulo.com.

(Versión en español.)

Available routes:

Name Route Example Notes
Book list /books /books —
Book by id /books/:id /books/11f2a45a-bfc5-41ad-af5d-0e482e84f87b —
Book search by title or author /books/search?q=<query>&by=<order-by>&order=<asc|desc> /books/search?q=lewis&by=author&order=desc The q parameter is required.
Page list /pages /pages —
Page by id /pages/:id /pages/b80a0417-abfc-4fe9-acda-c2e78edd1161 —
Page param search /pages/by-params?book-id=<book-id>&page-number=<page-number> /pages/by-params?book-id=e55dbeb9-d7bb-43b5-b22a-0bf2659d7e42&page-number=1 Both the book-id and the page-number query params are required.
