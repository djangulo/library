Biblioteca — API

Este es el API para la aplicación de biblioteca localizada en https://library.djangulo.com.

(English version.)

Rutas disponibles:

Nombre Ruta Ejemplo Notas
Listado de libros /books /books —
Libros por id /books/:id /books/864d0e7c-c309-4186-9737-f20f580a3d3c —
Búsqueda de libros por título o autor /books/search?q=<query>&by=<order-by>&order=<asc|desc> /books/search?q=lewis&by=author&order=desc El parámetro q es requerido.
Listado de páginas /pages /pages —
Páginas por id /pages/:id /pages/9a606356-0678-4478-ac4d-ff8bf84634e8 —
Búsqueda de páginas por parámetros /pages/by-params?book-id=<book-id>&page-number=<page-number> /pages/by-params?book-id=864d0e7c-c309-4186-9737-f20f580a3d3c&page-number=1 Ambos parámetros, book-id y page-number son requeridos.
